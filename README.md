# Hedera Vehicle Resale DApp

## Prerequisites

- Node.js ≥ 20
- npm

## Backend (Smart Contracts)

1. `cd contracts`
2. `npm install`
3. `node compileAndDeploy.js <ContractName.sol>`

## Frontend

1. `cd frontend/hedera-dapp`
2. `npm install`
3. `npm start`
4. Open`http://localhost:4200`

## Available Scripts

- `npm start`– start Angular 19.2.0 dev server
